- [ ] **6.1.1** `src/scripts/demo_live.py` реализован

  - Загрузка модели → перевод в eval mode → `torch.no_grad()`
  - Покадровый цикл: read frame → resize (224×224) → normalize → backbone → temporal step → StreamingDetector
- [ ] **6.1.2** Оверлей реализован:
  - **Рамка зелёная** (толщина 8px) когда `is_active=True`
  - **Рамка красная** (толщина 4px) когда `is_active=False`

  - **Score bar** горизонтальная полоска внизу: красный→зелёный по score

  - **Threshold line** на score bar (вертикальная пунктирная линия)
- [ ] **6.1.3** Управление клавишами:
  - `SPACE` — pause/resume

- [ ] **6.1.4** Режим записи `--output result.mp4`:
